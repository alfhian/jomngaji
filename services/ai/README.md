# AI Service

FastAPI microservice that exposes voice scoring capabilities for the JomNgaji ecosystem.

## Commands

- `uvicorn app.main:app --reload` – start development server.
- `pytest` – run unit tests.
- `ruff check app` – lint Python sources.

Dependencies are managed with `pyproject.toml`. Use your preferred tool (e.g. `pip`, `uv`, or `poetry`) to install them.
