# Backend Service

NestJS API providing authentication, user, and lesson management for the JomNgaji platform.

## Commands

- `npm install` – install dependencies (run from repository root or this folder).
- `npm run start:dev` – start development server.
- `npm run lint` – lint TypeScript sources with ESLint.
- `npm run test` – execute Jest unit tests.

Prisma schema is stored in `prisma/schema.prisma` and targets a PostgreSQL database.
